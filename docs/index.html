Current security posture

  • HTTPS for web traffic.
  • WSS for real-time transport.
  • WebRTC peer-to-peer collaboration path.
  • Diagrams are currently not stored on the server.

FAQ

Is Pumler free?
Core editing is free forever. Some advanced features may become paid later.
Is Pumler open source?
Not yet. Product feedback and requests are public through GitHub issues.
Does Pumler support sequence diagrams online?
Yes, via PlantUML with live preview.
Can I edit C4 diagrams online?
Yes, Structurizr syntax is supported today.
Do I need to install anything?
No. It works in the browser.
Where do I send feedback?
Use github.com/pumler/pumler/issues or email info@pumler.com.
