For Architects And Tech Leads

Collaborative text-to-diagram editor with live preview.

Pumler is an online workspace for PlantUML and Structurizr (C4): real-time collaboration, multiple syntaxes, rich export options, and fast editing ergonomics.

Core capabilities in one workflow

Everything below is available today and focused on real diagram authoring in teams.

Live collaboration

Edit one diagram together in real time instead of sharing stale screenshots and copies.

Multiple syntaxes

Use PlantUML and Structurizr in one editor. Mermaid support is next. Command palette makes switching fast when you move between sequence diagrams and C4 views.

Rich export controls

Export as SVG or PNG, copy image to clipboard, choose theme, toggle background, and use upscale presets including x2 and x4 for documentation-quality output.

Syntax highlight and code completion

Readable syntax colors plus inline suggestions keep editing fast and reduce small DSL mistakes.

Current security snapshot

  • HTTPS for web traffic.
  • WSS and WebRTC transport for collaboration.
  • Diagrams are currently not stored on the server.

FAQ

Is Pumler free?
Core editing is free forever. Some advanced capabilities may move to paid plans over time.
Is Pumler open source?
Not yet. Product feedback and feature requests are public via GitHub issues.
Does Pumler support C4 diagrams?
Yes. Structurizr syntax is supported today.
Does Pumler support sequence diagrams online?
Yes. PlantUML sequence diagrams are supported with live preview.
Do I need to install anything?
No. Pumler runs in the browser.
